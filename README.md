# Custom Pong Game

A fun Pong game where you can use your own images as paddles!

## Features

- 🎮 Classic Pong gameplay
- 🖼️ Drag & drop custom images for paddles
- 📊 Score tracking
- 🎯 Dynamic ball physics with paddle spin
- ✨ Beautiful gradient UI

## How to Play

1. Open `index.html` in your web browser
2. Drag and drop an image for Player 1's paddle (or click to select)
3. Drag and drop an image for Player 2's paddle (or click to select)
4. Click "Start Game"
5. Play!

## Controls

- **Player 1**: W (up) / S (down)
- **Player 2**: Arrow keys ↑ (up) / ↓ (down)
